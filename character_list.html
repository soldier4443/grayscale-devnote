Grayscale Devnote

Created by Tura

GRAY

  • 기본 캐릭터.
  • 기본 능력: 없음
  • 특수 능력: 데이터 급류(Data Flow) - 근처 적들을 제거함. Data 3개 소모. 준비 시간 0.5초. 쿨타임 5초

ICY

  • 주변을 냉각시키는 캐릭터.
  • 기본 능력: 냉각기(Cooler) - 주변의 오브젝트들의 속도가 15% 감소
  • 특수 능력: 절대 영도(Absolute Zero) - 모든 오브젝트의 기능 정지. Data 12개 소모. 준비 시간 1.5초. 지속 시간 2초, 쿨타임 10초

TIMY

  • 시간을 조작하는 캐릭터.
  • 기본 능력: 타임 패러독스(Time Paradox) - 죽었을 때 1회 부활
  • 특수 능력: 블링크(Blink) - 진행 방향으로 순간 이동. Data 2개 소모. 쿨타임 1.5초
