DATA

  • 일정 주기로 화면에 랜덤 생성
  • 움직임 없음
  • Player와 충돌 시 파괴됨 -> 데이터량 증가

VACCINE

  • 일정 주기로 화면에 랜덤 생성
  • 등장 시점의 플레이어 위치로 선형 이동
  • Player와 충돌 시 파괴됨 -> 데이터량 감소
  • 화면 밖으로 나갈 시 파괴됨

VIRUS(Player)

  • 게임 시작 시 화면 정가운데 생성
  • 플레이어의 터치로 이동
  • 중앙 버튼 클릭 시 일정 데이터를 사용해서 능력 발동
  • DATA와 충돌 시 데이터 획득
  • VACCINE과 충돌 시 데이터 감소

SYSTEM

  • 일정 시간마다 플레이어를 공격.
  • 공격 전에 공격 범위를 먼저 표시해 줌.
  • 공격 종류와 범위는 랜덤
